# 简介
这是一个学习设计模式案例的集合，每个案例都有对应的Main作简单测试。

代码参考于：[Javadoop - 设计模式也可以这么简单](https://www.javadoop.com/post/design-pattern)

设计模式分为三大类：创建型模式，结构型模式，行为型模式。
